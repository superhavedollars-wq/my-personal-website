Learning Object

编程学习

编程学习的目标不是马上成为专业程序员，而是能够维护自己的个人网站，把想法做成真实页面。

学习类HTML / CSS / JSGitHub Pages
返回知识库

学习内容

目前主要学习 HTML、CSS、JavaScript、Git、GitHub 和 GitHub Pages 部署。

实践方式

我把个人网站当成练习场：每新增一个页面、修改一个样式、整理一次链接，都是一次真实练习。

当前收获

我开始理解页面结构、样式复用、响应式布局和静态网站发布流程，也知道了上传文件时目录层级非常重要。

下一步

继续维护 个人网站，把学习内容变成可访问、可展示、可复盘的页面。
