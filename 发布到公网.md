# 让别人能打开、能搜索到的发布步骤

现在的 `index.html` 是本地文件，别人打不开，Google / 百度也搜不到。必须先发布到公网。

## 最推荐：GitHub Pages

1. 登录 GitHub，新建一个公开仓库，例如 `personal-website`。
2. 把本文件夹里的所有内容上传到仓库根目录：`index.html`、`styles.css`、`script.js`、`assets/`、`pages/`、`robots.txt`、`sitemap.xml`。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main`，目录选择 `/root`，保存。
5. 等几分钟后会得到一个网址，通常类似 `https://你的用户名.github.io/personal-website/`。
6. 把这个网址发给 Codex，用它更新 `robots.txt` 和 `sitemap.xml`。
7. 到 Google Search Console 提交 `sitemap.xml`，地址通常是 `https://你的用户名.github.io/personal-website/sitemap.xml`。

## 注意

- 搜索引擎不是立刻收录，通常需要几天到几周。
- 想更快被搜到，可以把公开网址发到社交平台、个人主页、公开笔记或其他已被搜索引擎收录的页面上。
- 页面里包含个人经历、心理状态和学业信息，公开前请确认这些内容可以被任何人看到。
