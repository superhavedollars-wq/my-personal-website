个人网站

这个网站是我的数字名片，也是我的个人档案馆。它用来展示学习、项目、作品、博客和第二大脑。

返回知识库

页面介绍

个人网站是我目前最重要的公开展示系统。它把学习记录、项目过程、作品结果、博客复盘和 AI 对话记录放到一个可访问的地方。

为什么做

我希望世界上有一个地方不用迎合别人，也不用急着证明自己。它只是安静地记录：我在学什么，做了什么，失败了什么，又重新开始了什么。

当前进度

网站已经完成首页、个人知识库、学习、项目、作品、博客、AI 对话记录等页面，也逐步补齐对象页和内部链接。

我的实践

我使用 HTML、CSS、JavaScript 和 GitHub Pages 搭建网站，用 Codex 协助修改结构、样式和链接，并把产出整理到 网站作品 中。

遇到的问题

主要问题是上传文件时容易把文件夹层级弄错，导致页面在错误路径；另一个问题是内容多了以后，需要更清楚的页面结构和链接关系。

下一步计划

继续优化对象页、完善 sitemap、增加 GEO 友好的页面描述，并准备 Day4 GEO 策略 和 Day5 展示。
